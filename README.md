## Run Locally

No build step required. To preview locally:

1. Clone the repo:

```powershell
git clone https://github.com/Kamogelo-Kedige/Portfolio-of-Kedige-Kamogelo.git
cd "Portfolio-of-Kedige-Kamogelo"
```

2. Open `index.html` in your browser, or serve it with a static server for best results:

```powershell
# Simple Python static server (python3 required)
python -m http.server 8000
# then open http://localhost:8000
```
